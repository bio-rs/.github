# bio-rs

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/bio-rs/bio-rs/blob/main/LICENSE)

Open source Rust/WASM tools for biological AI models.

Python is where many bio-AI models are born. `bio-rs` is where the tooling
around them becomes portable, inspectable, and easier to package outside
notebooks.

## Scope

`bio-rs` focuses on a portable input/runtime tooling layer for biological AI.

The goal is not to replace Python research workflows. The goal is to make the
tools around Python-born bio-AI models easier to package, inspect, and run
outside notebooks.
